## Что уже на странице
- `ThePreloader` — SVG-draw анимация ЭД-знака (`/img/logo-mark.svg`) + ЭЛИДОМ буквами, 1.8с min + window.load.
- `TheNav` — шапка ТОЛЬКО с обрезанным лого-знаком (без текста ЭЛИДОМ), мобильный бургер + оверлей-меню. Меню: О компании / Услуги / Портфолио / Сотрудничество. Контакты: телефон + Telegram.
- `HeroSection` — порт hero-1: eyebrow, заголовок «Потолки и свет премиум-класса.» (с оранжевой точкой), sub, 2 кнопки (Оставить заявку → **открывает BriefModal**, Смотреть портфолио → #portfolio), LED-cove линия + цифры 300+/15 000+.
- `BriefModal` — модалка (Teleport to body), открывается через composable `useBriefOpen()` / `openBrief()`. 3 шага из ТЗ: тип объекта (квартира/дом/коммерция) → дизайн-проект (да/нет + опц. файл) → имя+телефон → success. На мобайле — bottom-sheet, на ≥460px — центрированная карточка. POST в `/api/leads` пока mocked (TODO).
