# ЭЛИДОМ — баги и решения

> Что ломалось и как починили. Чтобы не наступать дважды.

## Шрифт Ablation: брендбук путал назначение
- **Симптом:** изначально я подключил Ablation как заголовочный, Evolventa как текстовый.
- **Правда:** наоборот. Заголовки — Evolventa, текст — Ablation.
- **Фикс:** `prototype/css/base.css` — `--font-head: Evolventa`, `--font-body: Ablation`.

## Composer падал на Windows-бинд-маунте vendor/
- **Симптом:** `Could not delete .../voku/helper/data:` — двоеточие в имени папки.
- **Причина:** NTFS на Windows не разрешает символ `:` в именах файлов/папок. Bind-mount vendor пишет на Windows-диск.
- **Фикс:** vendor → именованный Docker volume (`backend-vendor`). Тот же приём для `frontend/node_modules` (`frontend-node-modules`). См. `docker-compose.yml`.
- **Побочка:** именованный volume создаётся под root. Перед первым `composer install` нужен `chown` через `docker compose run --rm -u root <service>`.

## Ablation: hyphen-minus (U+002D) рендерится как «Г»
- **Симптом:** в маске телефона `+7 (916) 327-72-60` и в тексте «Дизайн-проект» дефисы выглядели как кириллическая «Г».
- **Причина:** шрифт Ablation не содержит глифа для U+002D HYPHEN-MINUS (обычный дефис ASCII), браузер падает на странный fallback.
- **Проверка:** в hero «премиум‑класса» используется `&#8209;` (U+2011 NON-BREAKING HYPHEN) и рендерится корректно. Значит U+2011 в Ablation есть, а U+002D — нет.
- **Фикс:** все дефисы в Ablation-тексте заменены на U+2011 (`‑`). Касается:
  - формата телефона `formatPhone` в `BriefModal.vue`
  - placeholder `+7 (___) ___‑__‑__`
  - «Дизайн‑проект» в заголовках/сводке
- **Правило на будущее:** в любом тексте, который рендерится Ablation (это `--font-body`), использовать только U+2011 для дефисов. Для тире — em-dash «—» (U+2014), которая в Ablation тоже есть.

## nuxi init интерактивный, не отрабатывает в exec -T
- **Симптом:** `npx nuxi init` спрашивал про оверрайд папки и про модули, висел.
- **Фикс:** флаги `--force -t minimal --no-install --no-gitInit`. Все вопросы выключаются явными флагами.

## logo.svg весит 800+ КБ
- **Причина:** в SVG зашиты 4 base64-PNG + 2 path-clipPath-прямоугольника + 2 mask. Векторных контуров букв там физически нет — только растровые слои с цветовыми/альфа-масками.
- **Подтверждение:** `grep -oE '<[a-zA-Z]+' logo.svg | sort | uniq -c` → 4 image, 2 path (оба — прямоугольные клипы 1390×1720 и 1390×1565).
- **Решение:** для шапки — `logo.png` напрямую. Для прелоадера-анимации — собственная векторная трассировка в `frontend/public/img/logo-mark.svg`. Извлечь PNG-слои из исходника можно скриптом на Python (см. bugs-and-fixes если понадобится повторить).
- **Полезный референс:** `logo-layer-3.png` (один из 4 слоёв) содержит плакат со всеми вариантами лого — лучший источник для трассировки. Регенерируется простым скриптом, разбирающим `<image xlink:href="data:image/png;base64,...">`.
